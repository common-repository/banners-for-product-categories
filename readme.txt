== Description ==

For showing banners for product categories. 
For each product category, two banners can be setted, one for wide screens and one for mobiles (for mobiles is in the premium version).

These banners can be shown in a category template or page with the shortcode [banner-prod-cat-webs].
See a working example on webplugins.nl.

If you find a bug, please report it and I will solve it very soon.
